# Custom Vulnerability Scanner

A lightweight Python-based vulnerability scanner implemented entirely in a single file, `Scanner.py`. It is designed to perform basic security assessments of web applications by identifying common vulnerabilities and security misconfigurations through automated checks.

The scanner provides a simple command-line interface, making it suitable for learning, experimentation, and authorized security assessments. Its single-file architecture keeps the project easy to understand, modify, and extend without requiring a complex setup.

The scanner can perform checks such as identifying missing security headers, detecting exposed resources, analyzing HTTP responses, inspecting server information, and testing for common web application weaknesses. Scan results are displayed in a clear and organized format to help users review potential security issues efficiently.

This project is intended for educational purposes and authorized penetration testing only. It should be used exclusively on systems for which you have explicit permission to perform security assessments.
